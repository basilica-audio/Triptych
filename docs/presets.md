# Factory presets

Eight factory presets ship with Triptych (`presets/factory/*.json`, embedded via BinaryData - see `docs/preset-system-notes.md` in `basilica-audio/nave`, the M2 preset system's pilot implementation, for the file format and wiring this repo copied). Every preset is a settings snapshot over the existing parameter set - no bespoke DSP beyond the v0.2.0 Knee, v0.3.0 Ratio-range/Range, and v0.4.0 Gate/per-band M/S additions. See `docs/design-brief.md` (v0.2.0 soft-knee), `docs/design-brief-v3-dynamics.md` (v0.3.0 upward compression/Range), and `docs/architecture.md` (v0.4.0 downward expansion/gating, issue #25, and per-band Mid/Side, issue #24) for the full research-derived rationale, and `docs/research-notes.md` for the sourced numbers each preset draws on.

**v0.4.0**: all eight presets gain the new per-band Gate parameter keys (`lowGateEnabled`/`lowGateThreshold`/`lowGateRatio`/`lowGateAttack`/`lowGateRelease` and the Mid/High equivalents) and the new per-band M/S parameter keys (`lowMidSideEnabled`/`lowSideThreshold`/`lowSideRatio` and the Mid/High equivalents), all at their neutral (off) v0.4.0 defaults - none of the eight presets engage gating or M/S, matching the "complete snapshot" convention the preset format already follows for prior additions (Knee, Range).

| Preset | Category | Intent |
|---|---|---|
| **Default** | Init | The shipped defaults, unchanged - the blank-slate starting point (moderate Low, density-leaning Mid, peak-control-leaning High). |
| **Density Glue** *(v0.3.0: now showcases upward compression)* | Master | Low/High keep their gentle downward "knit together" ratios (1.3:1 / 1.4:1); **Mid is now genuinely upward (0.7:1, was 1.2:1 downward)**, lifting presence-region content toward cohesion instead of only cutting its peaks. Range engaged on all three bands at 8 dB as a safety margin. Judge only in context, at light (1-2 dB) gain reduction/lift. |
| **Peak Control** | Master | High-threshold/higher-ratio "catch the excesses" philosophy (thresholds ~-10/-8/-8 dB, ratios ~5/4/4:1, near-hard 15% knee, fast attack) - only the loudest peaks are touched. |
| **Low-End Tighten** | Bus | Single-band-focused workflow: only the Low band compresses meaningfully (threshold -20 dB, ratio 3.5:1, slower ballistics); Mid/High left near-transparent (ratio 1.2:1). |
| **De-Harsh Highs** | Bus | Only the High band compresses meaningfully (threshold -22 dB, ratio 3:1, fast attack, smoothing 60% knee) - targets sibilance/cymbal buildup without touching the low end. |
| **Mastering Safety Ceiling** | Master | All bands near-transparent (ratio ~1.3:1, threshold -28 dB); the High-band limiter is engaged at its default -3 dB threshold as insurance, not sound-shaping, for a mix that's already well-balanced. |
| **Parallel-Style Density** *(v0.3.0: now showcases upward compression + Range)* | Master | **All three bands pushed to genuine upward compression (0.6:1, was 1.15:1 downward)** at the same deep (-38 dB) thresholds, for a more audible density/lift effect than Density Glue. **Range engaged on all three bands at 10 dB** - without it, this combination of deep threshold and strongly upward ratio could push full-scale peaks far louder than musically useful; Range is what keeps the extremity usable. +1.5 dB Output trim, unchanged. |
| **Hard Limiter Ceiling** | Master | High band pushed to an 8:1 ratio with a 0% (hard) knee, backstopped by the High-band limiter at -1 dB - the peak-control extreme with the limiter as a true safety net. |

All presets are research-derived starting points, not measurements against reference hardware - see `docs/design-brief.md`'s and `docs/design-brief-v3-dynamics.md`'s honesty sections. The six presets not called out above are otherwise unchanged from v0.2.0; their JSON files gain the new Range parameter keys at their neutral (off) v0.3.0 defaults purely to keep the "complete snapshot" convention the preset format already follows.
